# 📊 Supply Chain Management Dashboard (Power BI)

## 📌 Project Overview
The **Supply Chain Management Dashboard** is a data analytics project developed using **Microsoft Power BI** to analyze and visualize supply chain operations.

This dashboard transforms raw operational data into meaningful insights that help monitor performance, identify inefficiencies, and support data-driven decision-making.

It provides analysis across inventory management, supplier performance, logistics efficiency, order fulfillment, and operational costs using interactive visualizations.

---

## 🎯 Objectives
- Analyze supply chain data using Business Intelligence tools
- Build an interactive Power BI dashboard
- Monitor inventory levels and product demand
- Evaluate supplier performance and delivery efficiency
- Analyze transportation and logistics performance
- Identify operational cost trends for optimization

---

## 🛠 Tools & Technologies
- **Microsoft Power BI** — Data visualization & dashboard creation  
- **Power Query** — Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** — Calculated measures  
- **Excel / CSV Dataset** — Data source and preprocessing  

---

## 📂 Dataset Description

The dataset represents supply chain operations of a fashion and beauty company.

### Product Information
- Product Type  
- SKU  
- Price & Availability  
- Units Sold  
- Revenue Generated  

### Inventory Data
- Stock Levels  
- Order Quantities  
- Lead Time  

### Supplier Information
- Supplier Name & Location  
- Production Volume  
- Manufacturing Costs  
- Defect Rates  

### Logistics & Transportation
- Shipping Carriers  
- Transportation Modes  
- Shipping Costs  
- Delivery Routes & Time  

---

## 🔄 Data Preparation
Data preprocessing steps performed in Power BI:

1. Imported dataset into Power BI Desktop  
2. Removed duplicate and incomplete records  
3. Handled missing values  
4. Converted columns into appropriate data types  
5. Created calculated measures using DAX
